\maketitle
\noindent This document bundles every figure and table of the analysis. Supervised baselines are evaluated on all cases via 5-fold subject-level out-of-fold cross-validation; TiRex-2 is zero-shot (inherently held-out). Cross-dataset transfer (Fig.~6, Table~7) uses the external MOVER cohort. Inference is case-clustered bootstrap throughout.
